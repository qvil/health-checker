# Health Checker

Health Checker with Code Travelers!

## Demo

https://health-checker.qvil.now.sh

![demo.gif](demo.gif)

## Prerequisites

- Node.js >= 8.x (Recommended the latest version)
- .env : Refer to [.env.sample](.env.sample)

## Install

```
npm install
```

## Start

```
npm start
```
